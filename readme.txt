Template Sites for Offsprout gives you the ability to store all of your Offsprout page, section, and module templates in a single site and then retrieve and use them on any Offsprout-powered site.

== Description ==

= What Are Template Sites? =

Template Sites are WordPress sites that exist for the sole purpose of building Offsprout page, row, and module templates. They require the Offsprout Page Builder plugin and the premium Offsprout Pro plugin to be installed.

Once this plugin is installed on a site, other Offsprout-powered sites are then able to be configured to find and retrieve templates from that site.

== Installation ==

1. Install the Offsprout Page Builder plugin either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

2. Install Template Sites for Offsprout either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

3. Install the Offsprout Pro premium plugin by purchasing it on https://offsprout.com, then upload the files to your server at wp-content/plugins.

4. After activating Offsprout, Offsprout Pro, and Template Sites for Offsprout, create Offsprout templates as you would normally.

5. Connect other Offsprout-powered sites to the template site by entering the URL of the template site in the Site Settings of those Offsprout-powered sites.
